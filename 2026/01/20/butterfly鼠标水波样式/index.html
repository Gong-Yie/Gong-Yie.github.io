自定义调整

你可以通过修改配置来调整效果：

  • 修改颜色：在 CSS 中修改 rgba(64, 158, 255, 0.8) 的颜色值

  • 调整大小：修改 CSS 中 width 和 height 的最终值

  • 调整速度：修改 animation-duration 的值

  • 禁用特效：在浏览器控制台中输入 mouseEffect.disable()

这个特效会在鼠标点击时产生蓝色的水波纹扩散效果，逐渐消失。如果你想要更丰富的效果，可以启用多色模式或鼠标轨迹效果。
